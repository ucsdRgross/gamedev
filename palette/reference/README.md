# Your reference images

Drop PNG, JPEG or GIF files onto the **Recolour** tab in the app and they land here, so your
library survives between sessions. You can also just copy files into this folder — the app
lists it on startup.

Animated GIFs are recoloured in full: every frame, played back at the source's own timing.

The built-in reference art is *not* here. It is generated from `src/core/recolor/samples.js`
so the gallery is never empty and the repository carries no binary blobs
(ARCHITECTURE.md §12.4).
